## Default Permission

Allow controlling the embedded libmpv player.

#### This default permission set includes the following:

- `allow-mpv-load`
- `allow-mpv-command`
- `allow-mpv-get`
- `allow-mpv-set`
- `allow-mpv-stop`
- `allow-mpv-pip`
- `allow-mpv-viewport`
- `allow-mpv-fullscreen`
- `allow-mpv-transform`
- `allow-mpv-brightness`
- `allow-mpv-haptic`
- `allow-mpv-thumb`
- `allow-registerListener`
- `allow-removeListener`
