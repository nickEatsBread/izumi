## Default Permission

Allow launching an external Android video player.

#### This default permission set includes the following:

- `allow-play-external`

## Permission Table

<table>
<tr>
<th>Identifier</th>
<th>Description</th>
</tr>


<tr>
<td>

`extplayer:allow-play-external`

</td>
<td>

Enables the play_external command without any pre-configured scope.

</td>
</tr>

<tr>
<td>

`extplayer:deny-play-external`

</td>

</td>
</tr>
