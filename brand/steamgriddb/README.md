# Izumi — SteamGridDB covers

Six ready-to-upload cover variants for the Izumi entry on [SteamGridDB](https://www.steamgriddb.com/) (Steam + Steam Deck). Each folder is a **complete set** — all 10 accepted dimensions, exact-sized, static PNG, well under the size limits. SteamGridDB lets you upload multiple images per slot, so you can submit whichever variants you like.

`overview.png` shows all six side by side.

## Variants

| Folder | Look |
|---|---|
| `flat/` | Whirlpool + wordmark on a dark night-water gradient. The safe default. |
| `bold/` | Oversized whirlpool bleeding off-frame. Punchy in a library grid. |
| `bright/` | Full brand-gradient background with the ink logo. Loud, high-visibility. |
| `solid/` | Flat single-color indigo. Ultra-minimal. |
| `mark-only/` | Large whirlpool, no wordmark. Clean and iconic. |
| `light/` | White background, gradient mark. Bright outlier against dark libraries. |
